# Spotify Discography Data Pipeline & Search System (Python, API Integration)
Developed a Python-based data pipeline using the Spotify Web API to extract and structure artist discography data into a custom JSON dataset. The project processes album and track metadata, including album details, release years, and track-level information, enabling further analysis and search functionality.

The system incorporates multiple classic sorting algorithms (Bubble Sort, Insertion Sort, Quick Sort, and Merge Sort) to organise the dataset, alongside both binary and sequential search implementations for efficient data retrieval.

Additionally, the project includes basic visualisation of album artwork and keyword-based search functionality, demonstrating end-to-end data handling from API extraction to user-facing output.

Note: Due to changes in Spotify’s API access model and the introduction of stricter authentication/paywall limitations, parts of this project may no longer function as originally implemented.
## Known Limitations

This project was originally developed using the Spotify Web API for open data access. 
However, due to recent changes in Spotify's API policies and access restrictions, 
some functionality (such as data retrieval) may no longer work without approved API access.

The core logic, data processing pipeline, and algorithm implementations remain valid.
